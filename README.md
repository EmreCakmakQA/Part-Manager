# Part-Manager

This is a CRUD GUI app created with Python, Tkinter Library and SQLite. It allows a user to create PC component orders and update them, storing the information to a database managed via SQLite.

## Tech Stack
- Python 3.8.3
- Tkinter 8.6
- SQLite3
- VSCode
